3️⃣  ENABLE GITHUB ACTIONS (1 min)
   → Go to: "Actions" tab

   → Wait for green checkmark ✅

4️⃣  ENABLE GITHUB PAGES (2 min)
   → Go to: "Settings" tab
   → Click: "Pages" in left sidebar
   → Source: Deploy from a branch

   → Wait 1-2 minutes

5️⃣  GET YOUR FEED URL (1 min)

   → Open your feed URL in browser
   → Should see XML starting with:
      <?xml version="1.0" encoding="UTF-8"?>
      <listings>

7️⃣  ADD TO FACEBOOK (5 min)
   → Go to: business.facebook.com/commerce
   → Create Catalog → Vehicles

   → Currency: SEK
   → Upload!
